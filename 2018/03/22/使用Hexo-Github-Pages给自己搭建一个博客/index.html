前言

使用Coding Pages服务

由于总所周知的原因，Github在国内的访问很不稳定，由Github Pages支撑的博客自然也难以访问，coding位于国内，并且也支持Pages服务。
使用coding时，只需要将仓库名命名为你的用户名.coding.me，就可以了，其他操作和Giihub pages相同。
